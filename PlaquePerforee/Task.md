# Plaque Perforée

Etat : `Start`

## Description

Le projet consiste à faire le model d'une plaque perforée afin de recevoir des outils de rangement.

## Objectifs

Les objectifs sont de maitriser le geometry node
afin de ranger un peu mieux les outils en tout genre.
